Vision Assist

Vision Assist is an innovative application designed to help visually impaired individuals navigate their surroundings with greater confidence and independence. The system uses a combination of computer vision techniques and machine learning models to detect objects, recognize faces, read text, and provide audio feedback to users.

Key Features

  • Real-time object detection and identification
  • Text recognition and reading (OCR)
  • Facial recognition of saved contacts
  • Distance estimation for obstacle avoidance
  • Voice-controlled interface
  • Audio feedback through spatial sound

Technologies Used

This project leverages several cutting-edge technologies:

  • Python for the core application logic
  • TensorFlow and YOLO for object detection
  • PyTorch for facial recognition models
  • Tesseract OCR for text recognition
  • Google Text-to-Speech API for voice feedback
  • OpenCV for image processing

NBA Position Performance Study

A data science project analyzing the evolution of NBA positions and their statistical performance over the past 20 years. This study explores how the role of traditional positions has changed and identifies trends in player performance metrics.

Key Research Questions

  • How have traditional NBA positions evolved over time?
  • What statistical categories show the most significant changes per position?
  • How has the three-point revolution affected different positions?
  • Is position-less basketball reflected in the data?
  • Which positions provide the best value based on salary vs. performance?

Methodology

The project involved:

  • Data collection from NBA API and Basketball-Reference
  • Data cleaning and preprocessing
  • Statistical analysis of performance metrics
  • Time series analysis of position trends
  • Visualization of key findings
  • Machine learning models to predict future trends

Technologies Used

  • Python for data processing and analysis
  • Pandas and NumPy for data manipulation
  • Matplotlib, Seaborn, and Plotly for visualization
  • Scikit-learn for machine learning models
  • Jupyter Notebooks for documentation
